🔀 ONE Switch
Departments share the same floor but can't see each other — divided by invisible walls (tags)
🏠
Single building only Works within ONE physical network. Can't stretch to another building (site).
🏷️
Tags on packets Each packet gets a VLAN ID tag (like a coloured sticker) so the switch knows which floor it belongs to.
🔢
Max 4,094 VLANs Only ~4,000 "rooms" per building. Fine for a small company, tight for the cloud.

⬡ VXLAN Tunnel — Virtual L2 Overlay
🌍
Spans many buildings / cities Works across multiple physical sites, data centres, or cloud regions connected via the internet.
📦
Packets inside packets Wraps (encapsulates) L2 traffic inside UDP packets — like putting a local letter inside a courier envelope to ship it globally.
🔢
~16 million segments 24-bit VNI identifier. Designed for massive cloud environments with millions of tenants.
